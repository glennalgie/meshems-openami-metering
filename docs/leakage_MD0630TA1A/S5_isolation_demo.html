What this demonstrates. EMS upstream of the fault carry the residual current and see it; those downstream see almost nothing — so the boundary brackets the fault, and the highest magnitude marks the electrically closest node. The round triggers on change (ramp + correlated onset), not on absolute level, because standing residual current exists even without a fault. When two nodes read nearly the same level the fault sits between them: confidence collapses and the mesh alerts instead of isolating. Under reverse power flow the exporting node lowers its own AC threshold to 27 mA and confidence is de-rated — the case that phase-angle tracking would resolve. The sub-300 ms life-safety trip is never done here — it stays on the module's hardware fault line.
